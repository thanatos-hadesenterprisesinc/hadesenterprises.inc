# Getting started

## Clone repo
`git clone git@github.com:letItCurl/minimal_marketing your-new-marketing-website-name`

## Install dependencies
#### Install with NPM
`npm install`
#### Install with Yarn
`yarn`


## Start local server
##### Develop with NPM
`npm run start`
##### Develop with Yarn
`yarn start`

## Build static website for production
##### Build with NPM
`npm run build`
##### Build with Yarn
`yarn build`

## Form
To use the form, you could use those services:
